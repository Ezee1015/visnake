# Visnake

A simple snake game written in C with similar keybindings to vi

## Keybindings
- `j` --> Down
- `k` --> Up
- `h` --> Left
- `l` --> Right
- `<Escape>` --> Pause
- `q` --> Quit
- `r` --> Restart

## Requirements
- ncurses

## Compile

```bash
gcc visnake.c -o visnake -lncurses
```

## Run
```bash
./visnake
```
